# Beckhoff PLC CX2040 y Visual C++

Pruebas de concepto de conexión y manejo de un PLC Bekhoff tipo industrial CX2040, a través de una interfaz ADS y C++

## Requerimientos

- PLC CX2040
- Twincat 3
- Visual Studio 2019 C++

## Importante
Agregar al path del Visual Studio la libreria "C:\TwinCAT\AdsApi\TcAdsDll\Lib\TcAdsDll.lib" y "C:\TwinCAT\AdsApi\TcAdsDll\Include\*.h"

Los ejemplos son derivados de: 

https://infosys.beckhoff.com/english.php?content=../content/1033/tc3_adssamples_net/185255435.html&id=
